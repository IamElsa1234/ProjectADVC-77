What Is VS Code?

Visual Studio also known as VS code is an offline programming platform. Visual Studio Code is a streamlined code editor with support for development operations like debugging, task running, and version control. It aims to provide just the tools a developer needs.




How To Set Up VS Code?

Setting up VS Code is easy, to use it, we first have to make a new folder, then click on "File" option on the top left corner. Open Folder, then click on new file to make your programme on VS Code
